### 虚拟机
* 就是一个虚拟的计算机。它是一款软件，用来执行一系列虚拟计算机指令。
* java 虚拟机是一台执行java字节码的虚拟计算机，它拥有独立的运行机制，其运行的java字节码也未必由java语言编译而成
* 特点：
    * 一次编译，到处运行
    * 自动内存管理
    * 自动垃圾回收功能
